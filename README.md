# ECOTRACK

Mockup web/PWA mobile-first para validar una experiencia de avisos ambientales sin registro previo.

## Abrir localmente

```powershell
python -m http.server 8080
```

Luego abrir:

```text
http://127.0.0.1:8080
```

## Alcance actual

- Hero con video documental optimizado.
- Flujo de aviso en tres pasos: evidencia, ubicacion y envio.
- Contacto opcional al final para seguimiento.
- Mapa y reportes mock para validar UX.
- PWA estatica liviana, sin backend.

## Limitaciones

- No guarda datos reales.
- No sube archivos a storage.
- No envia emails ni WhatsApp.
- Los reportes son mock para validar direccion visual y flujo.
